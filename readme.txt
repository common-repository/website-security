Wordpress Site Security plugin takes care of your safety by avoiding third party apps or plugins to remotely communicate and by hiding login errors. 

== Description ==

Plugin blocks XML-RPC to keep away hackers and hide login errors for safety purposes.

Major features of this plugin include:

* To block XML-RPC which allow third party apps or plugins to communicate remotely.
* To hide login errors with a common message.

== Installation ==

* Upload the Wordpress Site Security plugin to the `/wp-content/plugins/` directory
* Activate the Wordpress Site Security plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to turn security On? =
* Once you activate the plugin it automatically turns on the security for your site.
